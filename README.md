# 中医、传承、骗子、传统文化
## 项目介绍：
------------
### 项目背景

这个方向是一个挑战，不带褒贬。<br>
关于中医，一直有很多话题，<br>
只是想尝试弄清楚大家一直在这个话题上“反”什么？<br>
“挺”什么？<br>
*"它"究竟是什么？*<br>
“它”到底发生了什么？<br>
和关于“它”未来有哪些可能？<br>

这里我们将尝试寻找其中的问题所在。

### 确定研究问题、策略及输出清单
问题 | 目标 | 实践策略 | 输出清单
--- | --- | --- | ---
